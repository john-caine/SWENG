eCook is an interactive budget-meal cookbook for students interested in a healthy way of living. It allows the user to load up recipes and follow step by step instructions on how to cook the meal.

eCook provides many options to aid with cooking including: audio/video walkthroughs and tutorials, text instructions, the ability to update recipes for any number of guests, multiple cooking timers, and modern graphics.
